[close] CDash Column Definitions
Site Name of the site sending the build.
Build Name Name of the current build.
Update Update was successful.
Update The CVS or SVN update contains locally modified files or an error occured during the update.
Cfg CMake Configuration was successful.
Cfg CMake Configuration failed.
Error Build has no compilation errors.
Error Build has compilation errors.
Warnings Build has no compilation warnings.
Warnings Build has compilation warnings.
Build Min Time required to build the current project in minutes.
Test Not Run All the tests were run.
Test Not Run Number of tests that were not run.
Test Fail All the tests are passing.
Test Fail Number of failing tests.
Test Pass Number of passed tests.
Test Pass Some tests were failing.
Test min Time to run all the tests in minutes.
Build Time Current starting time of the build.
The dashboard can be sorted by clicking on any of the column headings. Shift click can be used to add additional columns to the sorting process.
